## About The Project
### This project uses the deep learning convolutional neural network <b>U-Net-LSTM</b> to help cars recognize their lane along with <b>YOLOv8</b> to identify signs and vehicle owners.

## Built With

### Framework and library

* Pytorch
* Python
* Opencv
* Pyfirmata
* Socket
